Metabolomics

A good place to start would be the Metabolomics Innovation Centre at the University of Alberta and their list of software.

  • Biological Magnetic Resonance Data Bank Metabolomics — a repository for data from NMR spectroscopy on proteins, peptides, nucleic acids, and other biomolecules.
  • COLMAR (Complex Mixture Analysis by NMR, OSU) — includes tools for deconvoluting and querying 1D and 2D NMR spectra from complex mixtures.
  • ECMDB (E. coli Metabolome Database) — a curated resource containing data and metabolic pathway diagrams for Escherichia coli. Contains 19,294 NMR and MS spectra and over 3,000 metabolites. (Reference: T. Sajed et al. (2016) Nucleic Acids Res, 44(D1): D495–501).
  • Metabolomics GWAS Server — provides genome-wide association study data on the human metabolome, with ongoing feature updates.
  • M2IA — a web server for Microbiome and Metabolome data integration and analysis, supporting statistical and multivariate modeling workflows.
  • MetaboAnalyst — a web server for comprehensive metabolomic data analysis, visualization, and interpretation. (Reference: J. Xia et al. (2015) Nucleic Acids Res, 43(W1): W251–W257).
  • Workflow4Metabolomics — an open-source platform for computational metabolomics based on the Galaxy framework. (Reference: Giacomoni F. et al. (2015) Bioinformatics, 31:1493–1495).
