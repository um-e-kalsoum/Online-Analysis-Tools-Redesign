Transcriptomics

N.B. This is outside my area of expertise. Please advise about sites to recommend.

  • DOGMA — a web server for proteome and transcriptome quality assessment. The data of interest are evaluated based on a comparison with a core set of conserved protein domains and domain arrangements. Depending on the studied species, DOGMA offers precomputed core sets for different phylogenetic clades, including Bacteria and Archaea. (Reference: Kemena C et al. (2019) Nucleic Acids Res. 47(W1): W507–W510.)
  • Babelomics 5 — offers user-friendly access to a full range of methods covering (i) primary data analysis, (ii) a variety of tests for different experimental designs, and (iii) enrichment and network analysis algorithms for the interpretation of gene expression results. (Reference: Alonso R et al. (2015) Nucleic Acids Res. 43(W1): W117–W121.)
  • ErmineJ — performs analyses of gene sets in high-throughput genomics data such as gene expression profiling. Designed for biologists with minimal informatics background to identify pathways that are “doing something interesting.” (Reference: Gillis J et al. (2010) Nat Protoc. 5(6): 1148–1159.)
  • HPC-T-Annotator — a tool for de novo transcriptome homology annotation using HPC infrastructures. Configuration is done via a web interface, and the resulting data can be viewed in Python notebooks. (Reference: Arcioni L et al. (2024) BMC Bioinformatics. 25(1): 272.)
  • Cyber-T — for 2-dye data (cy3/cy5-labelled cDNA). Control+Experimental is for Affymetrix-based data with separate control and experimental arrays. (Reference: Kayala M.A. & Baldi P. (2012) Nucleic Acids Res. 40(W1): W553–W559.)
  • NIA Array Analysis Tool — for microarray data, providing features like false discovery rate adjustment, ANOVA-based expression pattern analysis, PCA visualization, and 3D clustering. (Reference: Sharov A.A. et al. (2005) Bioinformatics. 21: 2548–2549.)
  • TRAPID — an online tool for de novo transcriptome analysis, allowing functional and comparative analyses through a highly optimized pipeline. (Reference: Van Bel M. et al. (2013) Genome Biol. 14(12): R134.)
  • TRUFA — (TRAnscripome User-Friendly Analysis), a web-based open informatics platform for RNA-seq analysis and transcript quantification. Requires registration. (Reference: Kornobis E et al. (2015) Evol Bioinform Online. 11: 97–104.)
  • Chipster — a user-friendly software for analyzing high-throughput data such as NGS and microarrays, with over 360 analysis tools and reference genomes.
